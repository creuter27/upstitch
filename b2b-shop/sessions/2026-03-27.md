# Session 2026-03-27 — Project Kickoff

## Decisions Made

### Project Scope
- B2B Shopify store for reselling embroidered apparel (caps, aprons, etc.) from suppliers
- Products auto-scraped from supplier websites, AI-processed with embroidery mockups, listed automatically
- Customers can customize items (logo upload, yarn color, font, text) with realistic preview
- SaaS control panel for internal team to manage everything

### Architecture
- **SaaS backend**: FastAPI (Python) or Next.js API routes + PostgreSQL
- **SaaS frontend**: Next.js + Tailwind
- **Auth**: Supabase Auth or Auth.js
- **Scrapers**: Python + Playwright (one per supplier)
- **Image pipeline**: Pillow for quick preview, self-hosted Stable Diffusion for photorealistic AI inpainting
- **Shopify sync**: Shopify Admin API
- **Customization plugin**: Shopify Theme App Extension (Remix)
- **Hosting**: Vercel (frontend) + Railway/Fly.io (backend + scrapers)

### Suppliers
- utteam.com/de/catalog
- silvini.de
- Both to be ordered via email (lowest complexity); Silvini also supports Google Sheet

### Image Pipeline
- Listing mockups: "YOUR LOGO HERE" placeholder via SD inpainting
- Customer customizer: instant Pillow preview → on-demand AI render on "Preview" click
- AI: self-hosted Stable Diffusion (not Replicate)

### Product Workflow
- All scraped products require manual approval before going live in the shop
- No auto-listing

### Shop Design Direction
- Theme: Impulse or Expanse (Shopify paid themes)
- Style: "Clean Professional with a Craft Edge"
- Palette: off-white, dark charcoal, deep navy or forest green accent
- Typography: geometric sans (Inter/DM Sans) + warm serif for headlines
- Reference: engelbert-strauss.de (gold standard for B2B workwear UX)

### Session Logging
- `sessions/YYYY-MM-DD.md` created in every project
- Global Stop hook in `~/.claude/settings.json` auto-commits `sessions/` on Claude stop
